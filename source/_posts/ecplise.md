# Ecplise


## 常用快捷键(macOS)

* `cmd + shift + r ` 搜索资源
* `cmd + o` 查看当前类所有方法
* `cmd + e` 快速切换编辑文件
* `cmd + opt + r` 重命名
* `cmd + opt + l, cmd + opt + m` 提取本地变量及方法
* `shift + enter` 在下行创建空白行
* `opt + ↑/↓` 当前行上下移动
* `ctrl + m` 编辑器窗口最大化
* `cmd + ., cmd + 1` 快速定位错误，快速修改
* `ctrl + opt + h` 方法被哪些方法调用
* `cmd + t` 继承树
* `cmd + [/]` 回退阅读位置
* `cmd + shift + o` 导入包
